# 📱 RunSpot APK 설치 가이드

## 🎉 APK 빌드 완료!

RunSpot 안드로이드 APK가 성공적으로 빌드되었습니다. 이제 휴대폰에서 네이티브 전체화면 네비게이션을 테스트할 수 있습니다!

## 📍 APK 파일 위치

```
/Users/makingchae/runspot/android/app/build/outputs/apk/debug/app-debug.apk
```

**파일 크기**: 약 5.9MB

## 📲 휴대폰 설치 방법

### 방법 1: USB 케이블 연결
1. **USB 케이블로 휴대폰과 컴퓨터 연결**
2. **파일 전송 모드 활성화** (휴대폰에서 알림 확인)
3. **APK 파일을 휴대폰으로 복사**
   ```bash
   # 터미널에서 실행 (Android 디바이스가 연결된 경우)
   adb install android/app/build/outputs/apk/debug/app-debug.apk
   ```

### 방법 2: 파일 공유 (추천)
1. **APK 파일을 클라우드에 업로드**
   - Google Drive, Dropbox, iCloud 등
   - 또는 이메일로 자신에게 전송
2. **휴대폰에서 다운로드**
3. **파일 매니저에서 APK 파일 실행**

### 방법 3: QR 코드 (향후 구현)
- APK 파일을 웹 서버에 업로드 후 QR 코드로 공유

## ⚙️ 설치 전 준비사항

### 1. 개발자 옵션 활성화
1. **설정** → **휴대폰 정보** → **빌드 번호** 7번 연속 터치
2. "개발자가 되었습니다" 메시지 확인

### 2. 알 수 없는 소스 허용
1. **설정** → **보안** → **알 수 없는 소스** 활성화
2. 또는 **설정** → **앱 및 알림** → **특별한 앱 액세스** → **알 수 없는 앱 설치**

### 3. 위치 권한 준비
- GPS가 활성화되어 있는지 확인
- 위치 정확도를 "높음"으로 설정

## 🚀 앱 실행 및 테스트

### 1. 앱 설치 후 실행
1. **RunSpot** 앱 아이콘 터치
2. 초기 로딩 화면 확인
3. 위치 권한 요청 시 **허용** 선택

### 2. 네이티브 네비게이션 테스트
1. **코스 선택** (기존 웹 기능)
2. **런닝 시작** 버튼 클릭
3. **전체화면 네비게이션** 버튼 클릭
4. **네이티브 네비게이션 액티비티** 실행 확인

### 3. 확인할 기능들
- ✅ **플랫폼 자동 감지**: 안드로이드에서 네이티브 모드 자동 활성화
- ✅ **실시간 GPS 추적**: 위치 업데이트 확인
- ✅ **전체화면 UI**: 네이티브 네비게이션 화면 표시
- ✅ **네비게이션 계산**: 속도, 방향, 진행률 표시
- ✅ **웹-네이티브 통신**: 데이터 동기화 확인

## 🐛 문제 해결

### 설치 실패 시
1. **저장 공간 확인**: 최소 50MB 여유 공간 필요
2. **안드로이드 버전**: Android 6.0 (API 23) 이상 필요
3. **권한 확인**: 알 수 없는 소스 설치 허용 여부

### 앱 실행 실패 시
1. **위치 권한**: GPS 권한이 허용되었는지 확인
2. **네트워크**: 인터넷 연결 상태 확인
3. **로그 확인**: 개발자 도구에서 에러 로그 확인

### 네이티브 네비게이션 실행 안됨
1. **플랫폼 감지**: 웹 브라우저가 아닌 네이티브 앱에서 실행
2. **권한 확인**: 위치 권한이 허용되었는지 확인
3. **GPS 활성화**: 설정에서 GPS가 켜져 있는지 확인
